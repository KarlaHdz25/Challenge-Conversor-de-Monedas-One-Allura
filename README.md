# Challenge-Conversor-de-Monedas-One-Allura
El Conversor de Monedas es una aplicación de consola en Java que permite convertir valores entre distintas divisas en tiempo real. A través de un menú interactivo, el usuario elige un par de monedas, ingresa el monto que desea convertir, y la app consulta automáticamente la tasa de cambio actual a través de la API de ExchangeRate.
